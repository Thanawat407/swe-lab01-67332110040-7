### สรุปผลการทดลอง (README สรุป)
จากการทดลองนำโค้ดคำนวณเกรดภาษา Python ไปให้ AI ตรวจสอบหา Bug และทดสอบรันด้วย Test Case ต่างๆ สามารถสรุปได้ดังนี้:
1. **ผลการทำงานปกติ:** โค้ดสามารถคำนวณเฉลี่ยและตัดเกรด A-F ได้ถูกต้องเมื่อมีชุดตัวเลขคะแนนส่งเข้ามา
2. **Bug ที่พบ:** เกิดข้อผิดพลาด `ZeroDivisionError` (โปรแกรมพัง) เมื่อทดสอบด้วยกรณีขอบเขต (Edge Case) คือการส่งลิสต์ว่าง `[]` เข้าไป เนื่องจากตัวหาร (จำนวนคะแนน) กลายเป็น 0
3. **การประเมิน AI (Trust Level):** อยู่ในระดับ **4 เต็ม 5** เนื่องจาก AI สามารถตรวจจับและอธิบายสาเหตุของ Bug หลักๆ ได้อย่างรวดเร็ว แต่ยังคงต้องการการตรวจสอบและเพิ่มโค้ดป้องกันจากมนุษย์ในกรณีพิเศษ
